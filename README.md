# 두더지 잡기
## 메인 스토리
농부가 두더지 때문에 무척 속이 상해서 어느 날 해머를 들고 두더지를 잡으러 간다

## 개요
- 장르 : 퍼즐 / 보드
- 내용 : 땅굴 속에서 나오는 두더지를 잡아라
- 화면 크기 : 500*400
- 조작 방법 : 마우스
- 대상 : 모든 사용자
- 하드웨어 : PC
- 소프트웨어 : 웹 브라우저
- 기타 과제
    - 시간이 지남에 따라 난이도를 올린다.
    - 구멍의 수를 늘려본다.
    - 두더지 이외의 다른 캐릭터를 등장시킨다.
    - 다양한 효과음을 준비한다.
    - 망치 타격 효과를 변경한다.
